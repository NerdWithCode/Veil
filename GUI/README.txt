GUI tips: 
-Right click will deselect currently selected object
-pressing the 't' key will always reselect the tilemap

After creating the level that you want, go to file->Save as and follow the instuctions on the terminal.
Then, go into 'Veil 1.0 release->Data'. Paste your level into here and open 'Level list.dat' in any text editor, then write its name in the list. Save and exit the file, then boot up the game and your level should appear in the level select menu
